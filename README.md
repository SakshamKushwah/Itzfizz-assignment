# Scroll-Driven Hero Section Animation

Assignment implementation using:
- HTML
- CSS
- JavaScript
- GSAP + ScrollTrigger

## Features Implemented

1. Hero above the fold with spaced headline: `WELCOME ITZ FIZZ`
2. Intro load animation:
   - Eyebrow fades in
   - Headline letters stagger in
   - Stats cards animate one-by-one
3. Scroll-driven core animation:
   - Main hero visual transforms based on scroll progress (`scrub`)
   - Glow layer scales with scroll for depth
4. Performance-minded animation:
   - Transform-based motion
   - Minimal DOM work on scroll (handled by GSAP)

## Local Run

Open `index.html` in browser.
